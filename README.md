## Features

- Lightweight and fast, optimized for low-resource devices.
- Simple, responsive, and user-friendly interface.
- Built entirely with **Vue.js via CDN**, no build tools required.

---

## Technologies Used
- **Frontend:** Vue.js (via CDN), HTML, CSS
- **API:** CurrencyBeacon API

---

## How It Works
1. Select the source and target currencies.
2. Enter the amount to convert.
3. The app fetches live exchange rates from the **CurrencyBeacon API**.
4. The converted amount is displayed instantly.

---

## Why Vue CDN?
- Minimal setup, no Node.js or build tools required.
- Faster development on low-RAM devices.
- Easy deployment directly in any web browser.
