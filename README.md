# currency-converter
Currency Converter (Vue CDN Version)

A lightweight Currency Converter web application built using Vue.js via CDN, HTML, and CSS, powered by the CurrencyBeacon API. Designed to run smoothly even on low-RAM systems (like a 1â€¯GB Arch Linux laptop), this app allows users to quickly convert currencies in real-time without heavy frameworks or build tools.

Features

Real-time currency conversion using CurrencyBeacon API.

Lightweight and fast, optimized for low-resource devices.

Simple, responsive, and user-friendly interface.

Built entirely with Vue.js via CDN, no local build setup required.

Technologies

Frontend: Vue.js (via CDN), HTML, CSS

API: CurrencyBeacon API

How It Works

User selects source and target currencies.

User inputs the amount to convert.

The app fetches live exchange rates from CurrencyBeacon API.

Converted amount is displayed instantly.

Why Vue CDN

Using Vue via CDN ensures:

Minimal setup, no Node.js or build tools required.

Faster development for low-RAM devices.

Easy deployment directly in any web browser.
